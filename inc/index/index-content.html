Concept

我々は今回、全周囲ディスプレイを搭載した新しい携帯型ゲーム機”UPLIGHT”による、これまでにないゲーム体験を提案する。

立方体や半球などの立体構造物の全周囲にプロジェクションマッピングなどを用いて映像を表示させたものを全周囲ディスプレイという。映像が360°連続的に表示可能で、投影された立体形状にも直接手で触れることができるため、映像をより立体的に実感できる長所がある。

しかし、我々が「高さのある側面」を持つ立体映像を眺めたとき、我々に対して手前側の側面を見ることはできるが、反対側の側面は手前側の側面の背後に隠れて見えなくなってしまう。この「隠れた部分が見えない」という要素は全周囲ディスプレイの欠点の一つであった。

だが我々は、この欠点の中に「隠れた部分を見に行く」動作の持つ高いエンタテインメント性を見出した。プレイヤーからは常に見えない部分があり、そこを能動的に見に行くことで、見えなかった部分が明らかになる。これをゲーム要素に取り入れることで、プレイヤーに感動を与える新しいゲーム体験が実現できるのである。さらに、自由で楽しい動作で隠れた部分を身に行けるようにすることで、その体験の質を一段と高めることもできる。

そこで我々は、これらのゲーム体験を実現する新しい独自のゲームデバイス”UPLIGHT”を開発した。UPLIGHTでは内蔵する小型プロジェクタと超広角魚眼レンズを用いた全周囲投影技術に加え、自由に取り換えることができる様々な形状の全周囲スクリーンにより、立体的で連続的なゲームコンテンツを実現した。さらに手や頭などの上半身を自由に動かすことでダイナミックに全周囲ディスプレイ上のゲームと対話することができる。

UPLIGHTは完全に次世代のゲーム機であるが、これまでの携帯ゲーム機に親しんできたプレイヤーにとっても親しみやすく様々な人がこの体験に没入できるはずである。

Features

Replacable & Rotatable Omni-Directional Displays

Omni-Directional

UPLIGHTはプレイヤーから見えない部分にも映像を映し出す、全周囲ディスプレイを搭載したゲーム機である。

プレイヤーは見えない部分を見に行くため、上半身を動かして覗き込んだり、UPLIGHT自体をひっくり返して見ることになる。

Replacable

UPLIGHTの全周囲ディスプレイは手軽に取り換えることができ、その性質を活かした従来にないゲーム表現が可能である。

立方体形状、半球形状、円柱形状や正四面体形状など様々な形状のディスプレイを交換することで、更に遊びの幅を広げることができる。

Cube

Dome

Rotatable

UPLIGHTの立体全周囲ディスプレイは自動で回転し、または手動で回転させることができる。

手動回転

UPLIGHTの立体全周囲ディスプレイは手で持って回転させることができる。これにより、プレイヤーは細かい角度の制御を行うことができる。

また実際にディスプレイを手で持ち、思うがままに回転させられることで、プレイヤーに全周囲ディスプレイへの没入感を更に与えることができる。

自動回転

UPLIGHTの左右のコントローラには、それぞれ回転を制御するためのトリガーが1つずつ存在する。これにより、左右への自動回転を制御することができる。

またトリガーの押し込み具合によって、回転するスピードを調節できる。

Directional Speaker System

UPLIGHTのディスプレイ下部円周上には、外向きに8個のスピーカーが配置されている。

これにより、全周囲ディスプレイの側面部から直接音が出ているかのように聴こえ、立体全周囲ディスプレイへのプレイヤーの没入感をより高めることができる。

さらに、ゲーム機の裏側から聴こえてくる音によりプレイヤーは全周囲ディスプレイの隠れた部分へと更に強く誘導され、 「隠れた部分を見に行く」動作の持つエンタテインメント性をより高めることができる。

Controller

ディスプレイ左側に十字キーが、右側に決定ボタンがそれぞれ4つずつ存在する。これらは主にディスプレイに映るキャラクタの操作に用いる。

Contents

Breakout Game

従来のブロック崩しは、画面上を反射しながら移動するボールを、画面下部に落ちないように、バーを左右に操作して打ち返し、並べられたブロックを崩していくものである。

しかし、UPLIGHT上で展開されるブロック崩しは従来のものとは異なり、ボールはディスプレイの立体形状に沿う軌道を描く。 このため、これまで平面で遊んでいたプレイヤーは予想の斜め上をいくゲームを体験することになる。

また、ディスプレイの形状を取り換えることでボールの軌道が変わり、プレイヤーは形状毎に異なる体験が可能である。

Side-Scrolling Action Game

ステージが立方体ディスプレイ上で連続的に繋がり、キャラクタは隣り合う側面の間を行き来することができる。

プレイヤーはディスプレイを回転させながら、見えないステージにいるキャラクタやギミックを見に行くアクションを通して、UPLIGHTが提唱する「隠れた部分を見に行く」動作の持つ高いエンタテインメント性を体験できる。
